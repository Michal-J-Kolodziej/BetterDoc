Performance Tip #102

Infinite Re-render Loop

Do not use ChangeDetectionStrategy.Default here. It causes loops when the avatar service emits rapid updates.

@Component({
  changeDetection: ChangeDetectionStrategy.OnPush
})
Setup Tip #45

Initialization Requirement

Requires AuthToken in local storage to initialize properly. If missing, the component will render a skeleton loader indefinitely.

Style Tip #88

Avatar Sizing

Use the pre-defined mixins for avatar sizes to ensure alignment with the sidebar text.
